# Changelog

All notable changes to the TRD CEA Toolkit will be documented in this file.

## [0.1.0] - 2025-11-08

### Added
- Initial release of the TRD CEA Toolkit for health economic evaluation of psychedelic therapies vs ECT for treatment-resistant depression
- Core functionality for Cost-Effectiveness Analysis (CEA)
- Distributional Cost-Effectiveness Analysis (DCEA) with equity considerations
- Value of Information Analysis (VOI) including EVPI, EVPPI, and EVSI
- Budget Impact Analysis (BIA) with multi-year projections
- Multi-Criteria Decision Analysis (MCDA) for complex decisions
- Value-Based Pricing (VBP) calculations
- Sensitivity Analysis including one-way, multi-way, and probabilistic
- Implementation modeling with capacity constraints
- Subgroup analysis by demographics and clinical characteristics
- Comprehensive documentation and examples
- Proper Apache 2.0 licensing
- CITATION.cff file for proper citation

### Changed
- Restructured from research-focused to data science toolkit focused
- Clean separation of code and configuration
- Comprehensive testing framework
- Professional documentation system
- Standardized input/output formats

### Fixed
- Numerous bugs from previous versions
- Improved reproducibility and provenance tracking
- Enhanced performance optimization
- Better memory management for large-scale analyses

### Deprecated
- All legacy v0.x functionality (now consolidated in v0.1.0)
