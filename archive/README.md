# Archive Directory

This directory contains archived versions of the health economic evaluation project. The current canonical version is **V4**, located in the root directory.

## Archived Versions

### V1 Legacy (`v1_legacy/`)
**Status**: Baseline implementation
**Purpose**: Original health economic evaluation framework
**Key Features**: Basic cost-effectiveness analysis, simple PSA, initial manuscript materials
**Relationship to V4**: Provided foundation and baseline validation

### V2 Enhanced (`v2_enhanced/`)
**Status**: Enhanced analysis framework
**Purpose**: Comprehensive analysis capabilities with advanced features
**Key Features**: 
- Complete pipeline orchestration
- 11 analysis modules (CE plane, CEAF, BIA, tornado, VBP, price-prob, EVPI, etc.)
- 47 figure types and 25 data table types
- Core utilities framework
**Relationship to V4**: Provided proven analysis capabilities that V4 builds upon

### V3 NextGen (`v3_nextgen/`)
**Status**: Equity framework with publication enhancements
**Purpose**: Advanced equity analysis and publication-ready outputs
**Key Features**:
- Distributional cost-effectiveness analysis (DCEA)
- Equity impact assessment
- Publication-quality outputs (300 DPI, professional styling)
- Performance optimization and progress tracking
**Relationship to V4**: Provided equity analysis framework and publication standards that V4 incorporates

## Version Evolution

```
V1 (Baseline) → V2 (Enhanced) → V3 (Equity) → V4 (Canonical)
```

Each version built upon the previous, with V4 representing the comprehensive canonical implementation that consolidates all validated functionality while adding advanced features for publication.

## Accessing Archived Versions

Each archived version directory contains:
- Complete source code from that version
- Analysis results and outputs
- Version-specific documentation
- README explaining the version's features and purpose
